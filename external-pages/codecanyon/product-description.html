ExcelToWordAutoReport is a project developed with .Net Core with the C# language to generate reports from the Microsoft Office Interop library. The input data of the report is loaded from an excel file and the output format of the report can be chosen between Word and PDF.

You do not need to be a developer to use this application. In 5 minutes with the documentation step by step, you will learn everything you need to generate reports.

If you are a developer, this product generates an API, feel free to integrate it with any system or change the behavior of the report generator.
